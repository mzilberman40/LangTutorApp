**Input:** Native language, target language and dialect, list of phrases.
**Output:** Lesson files.

Lesson structure:
1. The length of each lesson is controlled by the `LESSON_LENGTH` parameter.
2. Each lesson consists of three files:
   - `lesson_NN.txt`: A text file containing the list of learning entities and phrases.
   - `lesson_NN_study.mp3`: An audio file for structured learning, where each phrase follows this sequence:
     1. The phrase is played in the native language.
     2. **`RECALL_PAUSE`**: A pause allowing the student to recall the phrase in the target language.
     3. The phrase is played in the target language for shadowing at the speed defined by `TARGET_AUDIO_SPEED`.
     4. **`REPETITION_PAUSE`**: The student repeats the phrase `REPETITION_COUNT` times.
     5. Steps 3 and 4 can be repeated `FULL_CYCLE_COUNT` times.
   - `lesson_NN_training.mp3`: A file for practicing the phrases of the lesson, where phrases appear in `TRAINING_ORDER`, requiring the student to recall and say them several times.

Audio files will be generated using a Python script, as previous AI-generated audio did not meet quality expectations.
The playback speed for native language phrases is controlled by `NATIVE_AUDIO_SPEED`, and for target language phrases by `TARGET_AUDIO_SPEED`.
