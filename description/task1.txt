1️ Пользователь вводит:
    1.words: список слов (например: apple, table, car).
    2.native_lang: родной язык. По умолчанию Русский.
    3.learning_lang: изучаемый язык. По умолчанию Британский английский
    4.level: Уровень изучаемого языка (A1, A2, ...., C2). По умолчанию B1
    5.Натуральное число N, по умолчанию 5

2 Отфильтровываются слова не принадлежащие изучаемому языку

3 Для каждого слова из оставшихся в списке:
    Составляется N наиболее употребительных фраз на изучаемом языке, соответствующие введенному уровню.

4 Генерируется json-файл со всеми фразами, получившимися на шаге 3.
    Количество json-объектов равно количеству фраз
    Каждый json-объект содержит следующие поля:
    native_lang, learning_lang, word, level
